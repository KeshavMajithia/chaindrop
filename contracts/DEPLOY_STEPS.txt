🔴 ERROR YOU'RE SEEING:
   Package object does not exist with ID 0xb3c59...

🔧 SOLUTION:
   The Sui Devnet was reset. You need to redeploy the contract.

═══════════════════════════════════════════════════════════════════

📋 STEP 1: Initialize Sui Wallet (First Time Only)
───────────────────────────────────────────────────────────────────

Run this command:
   sui client

When prompted:
   - "Connect to Sui Full node?" → Type: y
   - "Select network" → Choose: 0 (devnet)

═══════════════════════════════════════════════════════════════════

📋 STEP 2: Create Wallet Address (First Time Only)
───────────────────────────────────────────────────────────────────

Run this command:
   sui client new-address ed25519

⚠️  SAVE THE RECOVERY PHRASE SHOWN!

═══════════════════════════════════════════════════════════════════

📋 STEP 3: Get Testnet SUI Tokens
───────────────────────────────────────────────────────────────────

First, get your address:
   sui client active-address

Then get tokens from Discord:
   1. Join: https://discord.gg/sui
   2. Go to: #devnet-faucet channel
   3. Type: !faucet <your-address>
   4. Wait 30 seconds

Verify you have gas:
   sui client gas

═══════════════════════════════════════════════════════════════════

📋 STEP 4: Deploy Smart Contract
───────────────────────────────────────────────────────────────────

Navigate to contract folder:
   cd contracts\chaindrop_contracts

Build the contract:
   sui move build

Deploy to Devnet:
   sui client publish --gas-budget 100000000

═══════════════════════════════════════════════════════════════════

📋 STEP 5: Copy Package ID
───────────────────────────────────────────────────────────────────

Look for this in the output:
   PackageID: 0x1234567890abcdef...

Copy the entire Package ID (starts with 0x)

═══════════════════════════════════════════════════════════════════

📋 STEP 6: Update .env.local
───────────────────────────────────────────────────────────────────

Open: e:\chaindrop\.env.local

Add or update this line:
   NEXT_PUBLIC_SUI_PACKAGE_ID=0x<your_package_id_here>

Save the file.

═══════════════════════════════════════════════════════════════════

📋 STEP 7: Restart Dev Server
───────────────────────────────────────────────────────────────────

Stop the current server (Ctrl+C)

Then restart:
   npm run dev

═══════════════════════════════════════════════════════════════════

✅ DONE!
───────────────────────────────────────────────────────────────────

Your ChainDrop app should now work with Premium Mode!

Test it:
   1. Open http://localhost:3000/app
   2. Connect your Sui wallet
   3. Toggle "Premium Mode" ON
   4. Upload a file
   5. Sign the transaction
   6. Success! 🎉
